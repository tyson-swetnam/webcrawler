How It Works

AI University News Crawler is a production-grade Python application that automatically monitors university news sites for AI-related research announcements and developments. The system runs daily to deliver intelligent summaries of the latest artificial intelligence breakthroughs from leading research institutions.

The Crawling Process

The crawler operates through a sophisticated six-phase pipeline:

  1. Discovery & Crawling: Using Scrapy, the system visits designated university news pages and identifies new articles from press releases and news feeds.
  2. Content Extraction: Trafilatura extracts structured content from web pages with over 95% accuracy, capturing article text, metadata, publication dates, and authors.
  3. Deduplication: A PostgreSQL database combined with Bloom filters and SHA-256 hashing identifies truly new content, preventing duplicate articles from being processed.
  4. AI Analysis: Articles are analyzed in parallel using multiple AI APIs:
    • Claude Sonnet 4.5 (Anthropic) - Primary analysis with highest quality summaries
    • GPT-4 (OpenAI) - Secondary analysis and validation
    • Gemini 2.5 Flash (Google) - Initial AI-relevance filtering for cost optimization
  5. Reporting: The system generates comprehensive summaries and delivers them via multiple channels including this web interface, Slack webhooks, and email notifications.
  6. Persistence: All results are stored in a PostgreSQL database with complete logging for auditing and historical analysis.

Web Crawling Ethics

This crawler follows strict ethical guidelines:

  • Always respects robots.txt directives
  • Implements per-domain rate limiting (default: 1 request per second)
  • Uses descriptive User-Agent strings with contact information
  • Employs exponential backoff for failed requests
  • Stores crawl delays to ensure politeness

Technology Stack

  • Core: Python 3.11+ with Scrapy 2.11+
  • Content Extraction: Trafilatura 2.0+ with htmldate
  • Database: PostgreSQL 15+ for metadata/tracking, Redis 7+ for URL frontier
  • AI APIs: Claude (Sonnet 4.5), GPT-4, Gemini 2.5 Flash
  • Scheduling: Systemd timers or cron for automated daily execution
  • Notifications: Slack webhooks + SMTP email delivery
